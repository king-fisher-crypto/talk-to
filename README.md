# Setup

## Clone the repo
```shell
git clone git@gitlab.com:yann-srt/call_to.git
cd call_to
docker compose up -d
```

## Hosts
add the following line to your hosts file
```
127.0.0.1 talkto_php.local
```

Windows: C:\Windows\System32\drivers\etc\hosts
Linux: /etc/hosts
